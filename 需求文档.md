# 加密钱包在线查询网站 - 需求文档

## 项目概述
开发一个支持多链钱包地址查询的网站，提供余额、交易记录等信息查询服务，并优化SEO以获取自然流量。

## 支持的区块链网络

### 1. 以太坊 (Ethereum)
- ETH 主币余额查询
- ERC-20 代币余额查询
- 交易历史记录
- Gas 费用显示

### 2. 比特币 (Bitcoin)
- BTC 余额查询
- 交易历史记录
- UTXO 信息
- 交易确认数

### 3. 波场 (TRON)
- TRX 主币余额查询
- TRC-20 代币查询（重点：USDT-TRC20）
- 交易历史记录
- 能量和带宽信息

## 核心功能需求

### 1. 钱包查询功能
- **地址输入与验证**
  - 支持三种链的地址格式验证
  - 自动识别地址类型（ETH/BTC/TRX）
  - 支持扫码输入地址
  
- **余额显示**
  - 主币余额（ETH/BTC/TRX）
  - 代币余额列表
  - 实时价格转换（USD/CNY）
  - 资产总价值统计

- **交易记录**
  - 最近交易列表（分页）
  - 交易详情（时间、金额、手续费、状态）
  - 交易哈希链接到区块浏览器
  - 筛选功能（转入/转出/代币类型）

- **代币信息**
  - 代币名称、符号、数量
  - 代币价格和涨跌幅
  - 代币合约地址
  - 持仓价值

### 2. SEO 优化策略

#### 技术SEO
- **服务端渲染 (SSR)**
  - 使用 Next.js 实现 SSR
  - 预渲染静态页面
  - 动态路由支持（/eth/地址、/btc/地址、/trx/地址）

- **页面性能优化**
  - 首屏加载时间 < 2秒
  - 图片懒加载和压缩
  - 代码分割和按需加载
  - CDN 加速

- **移动端优化**
  - 响应式设计
  - 移动端友好的交互
  - 触摸优化
  - PWA 支持

#### 内容SEO
- **页面结构**
  - 语义化 HTML 标签
  - 合理的标题层级（H1-H6）
  - 结构化数据标记（Schema.org）
  - 面包屑导航

- **元数据优化**
  - 动态生成 Title 和 Description
  - Open Graph 标签
  - Twitter Card 标签
  - Canonical 标签

- **URL 结构**
  ```

  /guide/how-to-check-wallet # 教程页面
  /tools                     # 工具集合页
  ```

- **内容策略**
  - 每个查询页面生成独特的描述文本
  - 创建教程和指南页面
  - 常见问题 FAQ 页面
  - 区块链知识库
  - 定期更新的博客内容

## 技术架构建议

### 前端技术栈
- **框架**: Next.js 14+ (支持 SSR 和 SSG)
- **UI 库**: Tailwind CSS + shadcn/ui
- **状态管理**: Zustand 或 React Context
- **图表**: Chart.js 或 Recharts
- **请求库**: Axios 或 Fetch API

### 后端技术栈
- **运行时**: Node.js
- **框架**: Next.js API Routes 或 Express
- **缓存**: Redis（缓存查询结果，减少 API 调用）
- **数据库**: MongoDB（存储用户收藏、查询历史、缓存数据等）

### 区块链数据源

#### 以太坊
- Etherscan API
- Infura / Alchemy
- The Graph（可选）

#### 比特币
- Blockchain.com API
- Blockchair API
- Mempool.space API

#### 波场 (TRON)
- TronGrid API
- TronScan API
- 官方 TRON API

### 第三方服务
- **价格数据**: CoinGecko API / CoinMarketCap API
- **汇率转换**: ExchangeRate-API
- **分析工具**: Google Analytics / Plausible
- **SEO工具**: Google Search Console / Bing Webmaster

## 性能指标

### 技术指标
- Lighthouse 性能评分 > 90
- 首次内容绘制 (FCP) < 1.5s
- 最大内容绘制 (LCP) < 2.5s
- 累积布局偏移 (CLS) < 0.1

### SEO 指标
- Google 收录页面数
- 核心关键词排名
- 自然搜索流量
- 页面停留时间
- 跳出率

## 安全考虑

- 不存储用户私钥
- HTTPS 加密传输
- API 密钥安全存储
- 防止 XSS 和 CSRF 攻击
- 速率限制防止滥用
- 隐私声明和用户协议
