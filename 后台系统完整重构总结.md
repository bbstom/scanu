## 技术实现

### API 配置流程

1. **配置存储**: 每个代币的配置独立存储在 MongoDB
   - `API_CONFIG_ETH` - 以太坊配置
   - `API_CONFIG_BTC` - 比特币配置
   - `API_CONFIG_TRX` - 波场配置
   - `API_CONFIG_PRICE` - 价格配置

2. **配置读取**: `getApiConfig()` 从数据库读取配置
   - 过滤启用的平台
   - 按优先级排序
   - 返回平台列表

3. **API 调用**: `callApiWithFallback()` 依次尝试平台
   - 按优先级顺序调用
   - 失败自动切换到下一个
   - 所有平台失败才抛出错误

### 数据结构

```typescript
interface ApiPlatform {
  name: string;        // 平台名称
  url: string;         // API URL
  apiKey: string;      // API Key（可选）
  enabled: boolean;    // 是否启用
  priority: number;    // 优先级（1 最高）
}
```

### 以太坊服务实现

```typescript
// 获取余额
export async function getEthBalance(address: string): Promise<EthBalance> {
  return getCachedDataWithFallback(
    `eth:balance:${address}`,
    async () => {
      const balance = await callApiWithFallback("ETH", async (platform) => {
        // 根据平台选择不同的实现
        if (platform.name === "Etherscan") {
          return await getBalanceFromEtherscan(address, platform);
        } else if (platform.name === "Ethplorer") {
          return await getBalanceFromEthplorer(address, platform);
        }
        // ...
      });
      return { address, balance, balanceUSD: 0, tokens: [] };
    },
    60 // 缓存 60 秒
  );
}
```

## 使用指南

### 配置 API 平台

1. 登录后台 `/admin`
2. 自动进入以太坊 API 配置页面
3. 查看默认配置的平台
4. 修改配置：
   - 勾选/取消勾选启用
   - 修改优先级
   - 填写 API URL 和 Key
5. 点击"+ 添加平台"添加自定义平台
6. 点击"保存配置"

### 配置示例

**以太坊配置**:
```json
[
  {
    "name": "Etherscan",
    "url": "https://api.etherscan.io/api",
    "apiKey": "YOUR_API_KEY",
    "enabled": true,
    "priority": 1
  },
  {
    "name": "Ethplorer",
    "url": "https://api.ethplorer.io",
    "apiKey": "",
    "enabled": true,
    "priority": 2
  },
  {
    "name": "Blockscout",
    "url": "https://eth.blockscout.com/api",
    "apiKey": "",
    "enabled": true,
    "priority": 3
  }
]
```
