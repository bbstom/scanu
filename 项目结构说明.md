# 项目结构说明

## 目录结构

```
crypto-wallet-explorer/
├── src/
│   ├── app/                      # Next.js 14 App Router
│   │   ├── page.tsx             # 首页
│   │   ├── layout.tsx           # 根布局
│   │   ├── globals.css          # 全局样式
│   │   ├── robots.ts            # SEO: robots.txt
│   │   ├── sitemap.ts           # SEO: sitemap.xml
│   │   ├── eth/[address]/       # 以太坊地址查询页

│   │   ├── trx/[address]/       # 波场地址查询页（待实现）
│   │   └── api/                 # API 路由
│   │       └── search/          # 搜索记录 API
│   ├── components/              # React 组件
│   │   └── SearchBox.tsx        # 搜索框组件
│   ├── lib/                     # 工具函数
│   │   ├── mongodb.ts           # MongoDB 连接
│   │   └── redis.ts             # Redis 缓存
│   ├── models/                  # MongoDB 数据模型
│   │   └── SearchHistory.ts     # 搜索历史模型
│   ├── services/                # 区块链 API 服务
│   │   ├── ethereum.ts          # 以太坊服务
│   │   ├── bitcoin.ts           # 比特币服务
│   │   ├── tron.ts              # 波场服务
│   │   └── price.ts             # 价格服务
│   └── types/                   # TypeScript 类型定义
│       └── index.ts             # 通用类型
├── public/                      # 静态资源
├── .env.example                 # 环境变量示例
├── package.json                 # 项目依赖
├── tsconfig.json               # TypeScript 配置
├── tailwind.config.ts          # Tailwind CSS 配置

## 环境配置

### 必需的 API Key
1. Etherscan API Key - https://etherscan.io/apis
2. TronGrid API Key - https://www.trongrid.io/
3. CoinGecko API Key（可选）- https://www.coingecko.com/en/api

### 本地开发环境
1. Node.js 18+
2. MongoDB 6+
3. Redis 7+

## 注意事项

1. **API 限流**：免费 API 有请求限制，需要实现缓存
2. **数据准确性**：定期验证数据源的准确性
3. **安全性**：不存储用户私钥，仅查询公开数据
4. **性能**：使用 Redis 缓存减少 API 调用
5. **SEO**：持续更新内容，优化关键词排名
