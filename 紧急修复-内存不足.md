# 紧急修复 - Bus Error 内存不足

## 问题确认

你的服务器内存严重不足，即使使用优化配置也无法构建。

## 立即执行以下命令

### 步骤 1：检查当前内存状态

```bash
free -h
df -h
```

### 步骤 2：创建 Swap 空间（必须）

```bash
# 创建 2GB Swap
sudo fallocate -l 2G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile

# 验证
free -h
```

### 步骤 3：使用最小内存构建

```bash
cd /www/wwwroot/scanu.cc/scanu

# 清理
rm -rf .next node_modules/.cache

# 使用 256MB 内存构建
NODE_OPTIONS='--max-old-space-size=256' npm run build
```

### 步骤 4：如果还是失败，使用 128MB

```bash
NODE_OPTIONS='--max-old-space-size=128' npm run build
```

---

## 如果以上都失败 - 本地构建方案

### 在你的本地电脑上：

```bash
# 1. 进入项目目录
cd E:\vscodefile\usdtcheck

# 2. 安装依赖（如果还没安装）
npm install

# 3. 构建
npm run build

# 4. 打包构建文件
tar -czf build.tar.gz .next package.json package-lock.json public next.config.mjs
```

### 上传到服务器：

1. 在宝塔面板，点击 **文件**
2. 进入 `/www/wwwroot/scanu.cc/scanu`
3. 上传 `build.tar.gz`
4. 右键解压

### 在服务器上：

```bash
cd /www/wwwroot/scanu.cc/scanu

# 安装生产依赖
npm ci --production

# 启动
pm2 start npm --name crypto-app -- start
pm2 save
```

## 服务器配置检查

执行以下命令查看服务器配置：

```bash
# 查看内存
free -h

# 查看 CPU
nproc

# 查看磁盘
df -h

# 查看 Swap
swapon --show
```

如果显示：
- 内存 < 512MB：必须创建 Swap
- Swap = 0：必须创建 Swap
- 磁盘 < 2GB：需要清理空间

---

## 推荐方案：本地构建

由于你的服务器内存太小，最可靠的方案是：

1. ✅ 在本地电脑构建（Windows 有足够内存）
2. ✅ 上传构建文件到服务器
3. ✅ 服务器只负责运行（只需 200MB 内存）

这样可以避免所有构建问题！
