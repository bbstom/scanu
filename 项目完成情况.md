## 🚀 快速开始

### 三步启动

1. **安装依赖**：
```bash
npm install
```

2. **配置环境变量**（复制 .env.example 为 .env）：
```bash
cp .env.example .env
```

编辑 .env 文件，配置你的数据库地址和 API Keys：
```env
# 使用你自己的 MongoDB 数据库地址
MONGODB_URI=mongodb://你的数据库地址:27017/crypto-wallet-explorer

# 或使用 MongoDB Atlas
# MONGODB_URI=mongodb+srv://用户名:密码@cluster.mongodb.net/crypto-wallet-explorer

# API Keys
ETHERSCAN_API_KEY=你的key
TRONGRID_API_KEY=你的key
```

3. **启动开发服务器**：
```bash
npm run dev
```

4. 访问 http://localhost:3000

### 测试功能

访问 http://localhost:3000/test 可以测试各个 API 是否正常工作。

4. **免费使用**: 无需 API Key 即可正常运行
5. **测试**: 使用 /test 页面测试 API 是否正常

### 免费 API 支持

项目已内置多个免费公共 API：

**以太坊**:
- Ethplorer（免费，无需注册）
- Blockscout（免费，无需注册）

**比特币**:
- Blockchain.info（免费，无需注册）
- Blockchair（免费，每天 1440 次）
- Blockcypher（免费，每小时 200 次）

**波场**:
- TronScan（免费，无需注册）

**价格**:
- CoinGecko（免费，无需注册）

详细说明请查看：[免费API说明.md](./免费API说明.md)

### MongoDB 配置选项

- **本地数据库**: `mongodb://localhost:27017/crypto-wallet-explorer`
- **远程数据库**: `mongodb://服务器IP:27017/crypto-wallet-explorer`
- **MongoDB Atlas**: `mongodb+srv://用户名:密码@cluster.mongodb.net/crypto-wallet-explorer`
- **带认证**: `mongodb://用户名:密码@服务器:27017/crypto-wallet-explorer?authSource=admin`
