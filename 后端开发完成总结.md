## 数据库设计

### SearchHistory 集合

```javascript
{
  address: String,           // 钱包地址（索引）
  chainType: String,         // 链类型（索引）
  searchCount: Number,       // 搜索次数（索引）
  lastSearchAt: Date,        // 最后搜索时间（索引）
  createdAt: Date,          // 创建时间
  firstSearchAt: Date,      // 首次搜索时间
  tags: [String],           // 标签
  note: String,             // 备注
  isFavorite: Boolean,      // 是否收藏（索引）
  lastBalance: String,      // 最后余额
  lastBalanceUSD: Number,   // 最后余额(USD)
}
```

### 索引策略

```javascript
// 唯一索引
{ address: 1, chainType: 1 }

// 查询索引
{ searchCount: -1 }      // 热门地址
{ lastSearchAt: -1 }     // 最近搜索
{ isFavorite: 1 }        // 收藏夹
```

## Redis 缓存策略

### 缓存键设计

```
# 余额缓存
eth:balance:{address}
btc:balance:{address}
trx:balance:{address}

# 交易缓存
eth:transactions:{address}:{limit}
btc:transactions:{address}:{limit}
trx:transactions:{address}:{limit}

# 限流记录
ratelimit:{ip}

# 黑名单
ratelimit:blacklist

# 统计数据
stats:{period}
trending:{chain}:{period}:{limit}
```

### TTL 配置

| 类型 | TTL | 说明 |
|------|-----|------|
| 余额 | 60s | 1 分钟 |
| 交易 | 120s | 2 分钟 |
| 统计 | 300s | 5 分钟 |
| 限流 | 60s | 1 分钟 |
| 黑名单 | 3600s | 1 小时 |
