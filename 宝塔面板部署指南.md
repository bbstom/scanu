# 宝塔面板部署指南

## 前置准备

### 1. 安装宝塔面板
如果还没安装宝塔，执行：
```bash
# Ubuntu/Debian
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh

# CentOS
yum install -y wget && wget -O install.sh https://download.bt.cn/install/install_6.0.sh && sh install.sh
```

### 2. 登录宝塔面板
访问：`http://你的服务器IP:8888`

---

## 步骤 1: 安装必需软件

在宝塔面板中安装以下软件：

### 1.1 安装 Nginx
1. 点击左侧菜单 **软件商店**
2. 搜索 **Nginx**
3. 点击 **安装**（推荐 1.22 或更高版本）

### 1.2 安装 MongoDB
1. 在软件商店搜索 **MongoDB**
2. 点击 **安装**（推荐 6.0 或更高版本）
3. 安装完成后，点击 **设置**
4. 记下 MongoDB 的连接信息

### 1.3 安装 Redis（可选但推荐）
1. 在软件商店搜索 **Redis**
2. 点击 **安装**（推荐 7.0 或更高版本）

### 1.4 安装 Node.js
1. 在软件商店搜索 **Node 版本管理器**
2. 点击 **安装**
3. 安装完成后，点击 **设置**
4. 安装 **Node.js 18** 或更高版本

### 1.5 安装 PM2 管理器
1. 在软件商店搜索 **PM2 管理器**
2. 点击 **安装**

---

## 步骤 2: 创建网站

### 2.1 添加站点
1. 点击左侧菜单 **网站**
2. 点击 **添加站点**
3. 填写信息：
   - **域名**：你的域名（如 `crypto.example.com`）
   - **根目录**：`/www/wwwroot/crypto-wallet-explorer`
   - **PHP 版本**：选择 **纯静态**
   - **数据库**：不创建
4. 点击 **提交**

### 2.2 上传代码
有两种方式：

#### 方式 A: 使用 Git（推荐）
1. 点击站点右侧的 **设置**
2. 点击 **远程下载**
3. 输入你的 Git 仓库地址
4. 点击 **确定**

或者使用 SSH：
```bash
cd /www/wwwroot
git clone <你的仓库地址> crypto-wallet-explorer
cd crypto-wallet-explorer
```

#### 方式 B: 上传压缩包
1. 在本地打包项目（排除 node_modules 和 .next）
2. 在宝塔面板点击 **文件**
3. 进入 `/www/wwwroot/crypto-wallet-explorer`
4. 点击 **上传**，上传压缩包
5. 右键压缩包，点击 **解压**

---

## 步骤 3: 配置环境变量

### 3.1 创建 .env 文件
1. 在宝塔面板点击 **文件**
2. 进入 `/www/wwwroot/crypto-wallet-explorer`
3. 点击 **新建文件**，文件名为 `.env`
4. 编辑文件，填入以下内容：

```env
# MongoDB 配置
MONGODB_URI=mongodb://localhost:27017/crypto-wallet-explorer

# Redis 配置（如果安装了 Redis）
REDIS_URL=redis://localhost:6379
REDIS_ENABLED=true

# 管理员设置密钥（重要！请修改）
ADMIN_SETUP_KEY=your-secure-random-key-change-this

# 应用 URL（修改为你的域名）
NEXT_PUBLIC_APP_URL=https://crypto.example.com
NEXT_PUBLIC_BASE_URL=https://crypto.example.com

# 环境
NODE_ENV=production
```

### 3.2 配置 MongoDB 认证（可选但推荐）

如果需要 MongoDB 认证：

1. SSH 连接服务器
2. 进入 MongoDB：
```bash
mongo
```

3. 创建管理员用户：
```javascript
use admin
db.createUser({
  user: "admin",
  pwd: "your-secure-password",
  roles: ["root"]
})
```

4. 创建应用用户：
```javascript
use crypto-wallet-explorer
db.createUser({
  user: "crypto_user",
  pwd: "your-app-password",
  roles: ["readWrite"]
})
exit
```

5. 修改 `.env` 文件：
```env
MONGODB_URI=mongodb://crypto_user:your-app-password@localhost:27017/crypto-wallet-explorer?authSource=crypto-wallet-explorer
```

## 步骤 4: 安装依赖和构建

### 4.1 使用宝塔终端
1. 点击左侧菜单 **终端**
2. 进入项目目录：
```bash
cd /www/wwwroot/crypto-wallet-explorer
```

3. 安装依赖：
```bash
npm install
```

4. 构建项目：
```bash
npm run build
```

---

## 步骤 5: 使用 PM2 启动应用

### 5.1 在 PM2 管理器中添加项目
1. 点击左侧菜单 **软件商店**
2. 找到 **PM2 管理器**，点击 **设置**
3. 点击 **添加项目**
4. 填写信息：
   - **项目名称**：`crypto-wallet-explorer`
   - **启动文件**：选择 `node_modules/next/dist/bin/next`
   - **项目路径**：`/www/wwwroot/crypto-wallet-explorer`
   - **运行目录**：`/www/wwwroot/crypto-wallet-explorer`
   - **启动参数**：`start`
   - **端口**：`3000`
5. 点击 **提交**

### 5.2 或使用命令行
```bash
cd /www/wwwroot/crypto-wallet-explorer

# 启动应用
pm2 start npm --name "crypto-app" -- start

# 保存 PM2 配置
pm2 save

# 设置开机自启
pm2 startup
```

### 5.3 查看应用状态
在 PM2 管理器中可以看到：
- 应用状态（运行中/已停止）
- CPU 和内存使用
- 日志输出

## 步骤 6: 配置 Nginx 反向代理

### 6.1 配置站点
1. 点击左侧菜单 **网站**
2. 找到你的站点，点击 **设置**
3. 点击 **反向代理**
4. 点击 **添加反向代理**
5. 填写信息：
   - **代理名称**：`crypto-app`
   - **目标 URL**：`http://127.0.0.1:3000`
   - **发送域名**：`$host`
6. 点击 **提交**

### 6.2 高级配置（可选）
点击 **配置文件**，在 `location /` 块中添加：

```nginx
location / {
    proxy_pass http://127.0.0.1:3000;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection 'upgrade';
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;
    proxy_cache_bypass $http_upgrade;
    
    # 超时设置
    proxy_connect_timeout 60s;
    proxy_send_timeout 60s;
    proxy_read_timeout 60s;
}

# 静态文件缓存
location /_next/static {
    proxy_pass http://127.0.0.1:3000;
    expires 365d;
    add_header Cache-Control "public, max-age=31536000, immutable";
}
```

## 步骤 7: 配置 SSL 证书（HTTPS）

### 7.1 申请免费 SSL 证书
1. 在网站设置中，点击 **SSL**
2. 选择 **Let's Encrypt**
3. 填写邮箱
4. 勾选你的域名
5. 点击 **申请**

### 7.2 强制 HTTPS
1. 在 SSL 页面
2. 开启 **强制 HTTPS**

## 性能优化

### 1. 开启 Nginx 缓存
在网站设置 → 配置文件中添加：

```nginx
# 在 http 块中添加
proxy_cache_path /tmp/nginx_cache levels=1:2 keys_zone=my_cache:10m max_size=1g inactive=60m;

# 在 location / 块中添加
proxy_cache my_cache;
proxy_cache_valid 200 10m;
proxy_cache_key $scheme$proxy_host$request_uri;
```

### 2. 开启 Gzip 压缩
在网站设置 → 配置文件中确保有：

```nginx
gzip on;
gzip_vary on;
gzip_proxied any;
gzip_comp_level 6;
gzip_types text/plain text/css text/xml text/javascript application/json application/javascript application/xml+rss;
```

### 3. 调整 PM2 进程数
```bash
# 使用集群模式（多进程）
pm2 delete crypto-app
pm2 start npm --name "crypto-app" -i max -- start
pm2 save
```

## 安全建议

### 1. 修改宝塔默认端口
1. 点击 **面板设置**
2. 修改 **面板端口**（默认 8888）
3. 修改 **安全入口**

### 2. 开启防火墙
1. 点击 **安全**
2. 只开放必要端口：
   - 80（HTTP）
   - 443（HTTPS）
   - 22（SSH）
   - 8888（宝塔面板，可改）

### 3. 定期备份
1. 点击 **计划任务**
2. 添加任务：
   - **任务类型**：备份网站
   - **执行周期**：每天
   - **备份到**：本地或远程

### 4. 更新系统
定期在终端执行：
```bash
apt update && apt upgrade -y
```

## 故障排查

### 问题 1: 应用无法启动
1. 查看 PM2 日志：`pm2 logs crypto-app`
2. 检查端口是否被占用：`netstat -tulpn | grep 3000`
3. 检查 `.env` 文件配置是否正确

### 问题 2: 502 Bad Gateway
1. 检查应用是否运行：`pm2 status`
2. 检查端口 3000 是否监听：`netstat -tulpn | grep 3000`
3. 重启应用：`pm2 restart crypto-app`

### 问题 3: MongoDB 连接失败
1. 检查 MongoDB 是否运行：在宝塔面板查看 MongoDB 状态
2. 测试连接：`mongo mongodb://localhost:27017/crypto-wallet-explorer`
3. 检查 `.env` 中的连接字符串

### 问题 4: 内存不足
1. 在 PM2 管理器中查看内存使用
2. 增加服务器内存
3. 或限制 Node.js 内存：
```bash
pm2 delete crypto-app
pm2 start npm --name "crypto-app" --max-memory-restart 1G -- start
pm2 save
```
