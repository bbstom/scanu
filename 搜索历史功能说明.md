# 搜索历史功能说明

## 功能概述

系统提供了完整的搜索历史管理功能，包括历史记录、热门地址、收藏夹、标签管理等。

## 数据模型

### SearchHistory 字段

```typescript
{
  address: string;           // 钱包地址
  chainType: "eth" | "btc" | "trx";  // 链类型
  searchCount: number;       // 搜索次数
  lastSearchAt: Date;        // 最后搜索时间
  createdAt: Date;          // 创建时间
  firstSearchAt: Date;      // 首次搜索时间
  tags: string[];           // 自定义标签
  note: string;             // 备注
  isFavorite: boolean;      // 是否收藏
  lastBalance: string;      // 最后查询的余额
  lastBalanceUSD: number;   // 最后查询的美元价值
}
```

## API 端点

### 1. 记录搜索历史

```http
POST /api/search
Content-Type: application/json

{
  "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
  "chainType": "eth",
  "balance": "1.234567",
  "balanceUSD": 2345.67
}
```

**功能**:
- 记录搜索次数
- 更新最后搜索时间
- 保存余额信息

**限流**: 20 次/分钟

### 2. 获取搜索历史

```http
GET /api/history?type=recent&chain=eth&limit=10
```

**参数**:
- `type`: 类型
  - `recent`: 最近搜索（默认）
  - `popular`: 热门地址
  - `favorites`: 收藏夹
- `chain`: 链类型（可选）
  - `eth`: 以太坊
  - `btc`: 比特币
  - `trx`: 波场
- `limit`: 返回数量（默认 10）

**响应**:
```json
{
  "histories": [
    {
      "_id": "...",
      "address": "0x742d35Cc...",
      "chainType": "eth",
      "searchCount": 15,
      "lastSearchAt": "2024-03-01T10:30:00Z",
      "isFavorite": true,
      "tags": ["交易所", "热钱包"],
      "note": "Binance 热钱包",
      "lastBalance": "1234.567890",
      "lastBalanceUSD": 2345678.90
    }
  ]
}
```

### 3. 更新搜索历史

```http
PATCH /api/history
Content-Type: application/json

{
  "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
  "chainType": "eth",
  "updates": {
    "isFavorite": true,
    "tags": ["交易所", "热钱包"],
    "note": "Binance 热钱包"
  }
}
```

**可更新字段**:
- `isFavorite`: 收藏状态
- `tags`: 标签数组
- `note`: 备注

### 4. 删除搜索历史

```http
DELETE /api/history?address=0x742d35Cc...&chain=eth
```

### 5. 搜索建议（自动完成）

```http
GET /api/history/suggestions?q=0x742&chain=eth&limit=5
```

**参数**:
- `q`: 搜索关键词（至少 3 个字符）
- `chain`: 链类型（可选）
- `limit`: 返回数量（默认 5）

**响应**:
```json
{
  "suggestions": [
    {
      "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb",
      "chainType": "eth",
      "searchCount": 15,
      "lastSearchAt": "2024-03-01T10:30:00Z",
      "tags": ["交易所"]
    }
  ]
}
```

### 6. 热门地址

```http
GET /api/history/trending?chain=eth&period=24h&limit=10
```

**参数**:
- `chain`: 链类型（可选）
- `period`: 时间范围
  - `24h`: 最近 24 小时
  - `7d`: 最近 7 天
  - `30d`: 最近 30 天
  - `all`: 全部时间（默认）
- `limit`: 返回数量（默认 10）

**缓存**: 5 分钟

**响应**:
```json
{
  "trending": [
    {
      "address": "0x742d35Cc...",
      "chainType": "eth",
      "searchCount": 156,
      "lastSearchAt": "2024-03-01T10:30:00Z",
      "lastBalance": "1234.567890",
      "lastBalanceUSD": 2345678.90,
      "tags": ["交易所"]
    }
  ]
}
```

## 使用场景

### 1. 最近搜索

显示用户最近查询的地址：

```typescript
const response = await fetch('/api/history?type=recent&limit=5');
const { histories } = await response.json();
```

### 2. 热门地址

显示全站热门地址：

```typescript
const response = await fetch('/api/history/trending?period=24h&limit=10');
const { trending } = await response.json();
```

### 3. 收藏夹

管理用户收藏的地址：

```typescript
// 添加到收藏
await fetch('/api/history', {
  method: 'PATCH',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    address: '0x742d35Cc...',
    chainType: 'eth',
    updates: { isFavorite: true }
  })
});

// 获取收藏列表
const response = await fetch('/api/history?type=favorites');
const { histories } = await response.json();
```

### 4. 标签管理

为地址添加标签：

```typescript
await fetch('/api/history', {
  method: 'PATCH',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    address: '0x742d35Cc...',
    chainType: 'eth',
    updates: {
      tags: ['交易所', '热钱包', 'Binance'],
      note: 'Binance 主要热钱包地址'
    }
  })
});
```

### 5. 搜索自动完成

实现地址输入自动补全：

```typescript
const handleSearch = async (query: string) => {
  if (query.length < 3) return;
  
  const response = await fetch(`/api/history/suggestions?q=${query}&limit=5`);
  const { suggestions } = await response.json();
  
  // 显示建议列表
  setSuggestions(suggestions);
};
```

## 前端集成示例

### React 组件示例

```typescript
import { useState, useEffect } from 'react';

function SearchHistory() {
  const [histories, setHistories] = useState([]);
  const [type, setType] = useState('recent');

  useEffect(() => {
    fetchHistories();
  }, [type]);

  const fetchHistories = async () => {
    const response = await fetch(`/api/history?type=${type}&limit=10`);
    const data = await response.json();
    setHistories(data.histories);
  };

  const toggleFavorite = async (address: string, chainType: string, isFavorite: boolean) => {
    await fetch('/api/history', {
      method: 'PATCH',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        address,
        chainType,
        updates: { isFavorite: !isFavorite }
      })
    });
    fetchHistories();
  };

  return (
    <div>
      <div className="tabs">
        <button onClick={() => setType('recent')}>最近搜索</button>
        <button onClick={() => setType('popular')}>热门地址</button>
        <button onClick={() => setType('favorites')}>收藏夹</button>
      </div>

      <ul>
        {histories.map((item) => (
          <li key={item._id}>
            <span>{item.address}</span>
            <span>{item.chainType.toUpperCase()}</span>
            <span>搜索 {item.searchCount} 次</span>
            <button onClick={() => toggleFavorite(item.address, item.chainType, item.isFavorite)}>
              {item.isFavorite ? '取消收藏' : '收藏'}
            </button>
          </li>
        ))}
      </ul>
    </div>
  );
}
```

### 搜索建议组件

```typescript
function SearchInput() {
  const [query, setQuery] = useState('');
  const [suggestions, setSuggestions] = useState([]);

  const handleChange = async (value: string) => {
    setQuery(value);
    
    if (value.length >= 3) {
```

## 数据库索引

为了优化查询性能，已创建以下索引：

```javascript
// 唯一索引
{ address: 1, chainType: 1 }

// 查询索引
{ searchCount: -1 }      // 热门地址
{ lastSearchAt: -1 }     // 最近搜索
{ isFavorite: 1 }        // 收藏夹
```

## 性能优化

### 1. 缓存策略

- 热门地址缓存 5 分钟
- 减少数据库查询

### 2. 分页查询

```typescript
// 使用 limit 限制返回数量
const histories = await SearchHistory.find()
  .sort({ lastSearchAt: -1 })
  .limit(20)
  .skip(page * 20);
```

### 3. 字段选择

```typescript
// 只返回需要的字段
.select('address chainType searchCount lastSearchAt')
```

## 最佳实践

### 1. 定期清理

建议定期清理旧的搜索记录：

```javascript
// 删除 90 天前的非收藏记录
const ninetyDaysAgo = new Date(Date.now() - 90 * 24 * 60 * 60 * 1000);
await SearchHistory.deleteMany({
  lastSearchAt: { $lt: ninetyDaysAgo },
  isFavorite: false
});
```

### 2. 标签规范
