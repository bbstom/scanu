## 使用指南

### 配置以太坊 API

1. 登录后台（自动进入以太坊配置页面）
2. 查看默认配置的平台：
   - Etherscan（优先级 1）
   - Ethplorer（优先级 2）
   - Blockscout（优先级 3）
3. 修改配置：
   - 勾选/取消勾选启用平台
   - 修改优先级数字
   - 填写 API URL 和 API Key
4. 点击"+ 添加平台"添加自定义平台
5. 点击"保存配置"保存更改

### 配置其他代币

1. 在左侧菜单展开"API 配置"
2. 点击对应的代币（BTC/TRX/价格查询）
3. 按照相同方式配置平台
4. 每个代币的配置独立保存

### 查看数据统计

1. 在左侧菜单点击"数据统计"
2. 选择时间范围（24h/7d/30d/全部）
3. 查看概览、链统计、热门地址
