# 开发指南

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 配置环境变量

复制 `.env.example` 为 `.env`：

```bash
cp .env.example .env
```

编辑 `.env` 文件，填入以下配置：

```env
# MongoDB（必需）
MONGODB_URI=mongodb://localhost:27017/crypto-wallet-explorer

# Redis（可选，用于缓存）
REDIS_URL=redis://localhost:6379

# API Keys（必需）
ETHERSCAN_API_KEY=你的_etherscan_api_key
TRONGRID_API_KEY=你的_trongrid_api_key

# 价格 API（可选）
COINGECKO_API_KEY=你的_coingecko_api_key

# 应用配置
NEXT_PUBLIC_APP_URL=http://localhost:3000
NODE_ENV=development
```

### 3. 获取 API Keys

#### Etherscan API Key
1. 访问 https://etherscan.io/
2. 注册账号并登录
3. 进入 API Keys 页面：https://etherscan.io/myapikey
4. 创建新的 API Key（免费版每秒 5 次请求）

#### TronGrid API Key
1. 访问 https://www.trongrid.io/
2. 注册账号
3. 在控制台创建 API Key（免费版每天 15000 次请求）

#### CoinGecko API（可选）
- 免费版无需 API Key，但有请求限制
- 如需更高限制，访问 https://www.coingecko.com/en/api 购买 API Key

### 4. 启动数据库

确保你的 MongoDB 数据库服务正在运行。

如果使用本地 MongoDB：
```bash
# 启动 MongoDB 服务
mongod
```

如果使用远程数据库或 MongoDB Atlas，确保：
- 数据库服务正常运行
- 网络连接正常
- 防火墙允许连接
- IP 地址在白名单中（如果使用 MongoDB Atlas）

### 5. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

```
│   ├── page.tsx           # 首页

```

## API 使用说明

### 以太坊 API (Etherscan)

```typescript
import { getEthBalance, getEthTransactions } from "@/services/ethereum";

// 获取余额
const balance = await getEthBalance(address);

// 获取交易记录
const transactions = await getEthTransactions(address, page, limit);
```

### 比特币 API (Blockchain.com)

```typescript
import { getBtcBalance, getBtcTransactions } from "@/services/bitcoin";

// 获取余额
const balance = await getBtcBalance(address);

// 获取交易记录
const transactions = await getBtcTransactions(address, limit);
```

### 波场 API (TronGrid)

```typescript
import { getTrxBalance, getTrxTransactions } from "@/services/tron";

// 获取余额
const balance = await getTrxBalance(address);

// 获取交易记录
const transactions = await getTrxTransactions(address, limit);
```

### 价格 API

```typescript
import { getEthPrice, getBtcPrice, getTrxPrice } from "@/services/price";

const ethPrice = await getEthPrice();
const btcPrice = await getBtcPrice();
const trxPrice = await getTrxPrice();
```

## 缓存策略

使用 Redis 缓存查询结果，减少 API 调用：

```typescript
import { getCachedData, setCachedData } from "@/lib/redis";

// 获取缓存
const cached = await getCachedData<EthBalance>(`eth:${address}`);
if (cached) return cached;

// 设置缓存（TTL: 300秒）
await setCachedData(`eth:${address}`, data, 300);
```

## 常见问题

### API 请求失败

1. 检查 API Key 是否正确
2. 检查网络连接
3. 查看 API 请求限制是否超出
4. 查看控制台错误信息

### MongoDB 连接失败

1. 确保 MongoDB 服务已启动
2. 检查 MONGODB_URI 配置是否正确
3. 检查网络连接和防火墙设置
4. 如果使用 MongoDB Atlas，检查 IP 白名单
5. 查看详细错误日志

### 页面加载慢

1. 启用 Redis 缓存
2. 优化 API 调用
3. 使用 CDN 加速静态资源

## 部署

### Vercel（推荐）

1. 推送代码到 GitHub
2. 在 Vercel 导入项目
3. 配置环境变量（包括 MONGODB_URI）
4. 自动部署

注意：确保 MongoDB 数据库可以从外网访问，推荐使用 MongoDB Atlas。
